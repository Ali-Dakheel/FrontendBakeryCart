# Next.js: ALWAYS read docs before coding

Before any Next.js work, find and read the relevant doc. Your training data is outdated — the docs are the source of truth.

**Where to find docs (in order of preference):**

1. `node_modules/next/dist/docs/` — bundled, version-matched (available after upgrading to Next.js v16.2.0+)
2. GitHub raw fallback (same content, always current):
   - App Router: `https://raw.githubusercontent.com/vercel/next.js/canary/docs/01-app/`
   - API Reference: `https://raw.githubusercontent.com/vercel/next.js/canary/docs/01-app/03-api-reference/`
   - Guides: `https://raw.githubusercontent.com/vercel/next.js/canary/docs/01-app/02-guides/`
